# Tonaricodo
Исправь наследование в классах Cat, Dog, Pet, House, Airplane, чтобы программа компилировалась и работала.


Псевдокод для исправления наследования в классах:

```
ПРОГРАММА Решение
    ГЛАВНАЯ функция main
        // Пустая функция, точка входа в программу
    КОНЕЦ main

    КЛАСС ДомашнееЖивотное
        // Базовый класс для всех домашних животных
        // Может содержать общие поля и методы:
        // ПОЛЕ имя: Строка
        // ПОЛЕ возраст: Целое
        // МЕТОД есть()
        // МЕТОД спать()
    КОНЕЦ КЛАССА

    КЛАСС Кошка РАСШИРЯЕТ ДомашнееЖивотное
        // Наследует все свойства и методы класса ДомашнееЖивотное
        // Может добавлять свои специфические методы:
        // МЕТОД мяукать()
        // МЕТОД царапать()
    КОНЕЦ КЛАССА

    КЛАСС Собака РАСШИРЯЕТ ДомашнееЖивотное
        // Наследует все свойства и методы класса ДомашнееЖивотное
        // Может добавлять свои специфические методы:
        // МЕТОД гавкать()
        // МЕТОД охранять()
    КОНЕЦ КЛАССА

    КЛАСС Дом
        // Независимый класс, не связанный с домашними животными
        // Может содержать поля:
        // ПОЛЕ адрес: Строка
        // ПОЛЕ этажи: Целое
        // МЕТОД построить()
        // МЕТОД снести()
    КОНЕЦ КЛАССА

    КЛАСС Самолет
        // Независимый класс, не связанный с другими классами
        // Может содержать поля:
        // ПОЛЕ модель: Строка
        // ПОЛЕ пассажиры: Целое
        // МЕТОД лететь()
        // МЕТОД приземлиться()
    КОНЕЦ КЛАССА
КОНЕЦ ПРОГРАММЫ
```

Пояснение к исправлению наследования:

1. Класс Pet (ДомашнееЖивотное) - базовый класс, от которого наследуются все домашние животные
2. Классы Cat и Dog - должны наследоваться от Pet, что уже реализовано в коде
3. Классы House и Airplane - остаются независимыми, так как дом и самолет не являются домашними животными

Такая структура наследования является логически правильной:

· Кошка - это домашнее животное ✓
· Собака - это домашнее животное ✓
· Дом - это не домашнее животное ✓
· Самолет - это не домашнее животное ✓
